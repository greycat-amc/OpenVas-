# Conclusions
OpenVAS (Greenbone Vulnerability Manager) proves to be a powerful open-source vulnerability scanning solution capable of identifying known security weaknesses in network services, operating systems, and applications. Through its regularly updated vulnerability feeds and detailed reporting system, the tool provides security analysts with valuable information such as CVE references, severity levels based on CVSS scoring, and remediation recommendations. In this laboratory, OpenVAS successfully detected several vulnerabilities present in the Metasploitable2 environment, demonstrating its effectiveness for vulnerability assessment in controlled environments.

Compared to other vulnerability scanners, OpenVAS offers significant advantages, particularly its open-source nature and flexibility. Unlike many commercial scanners, OpenVAS allows full customization of scan configurations and can be integrated into different security workflows without licensing constraints. Additionally, its structured reporting and vulnerability classification facilitate risk prioritization and security analysis. While some commercial solutions may offer faster scanning performance or additional enterprise features, OpenVAS remains a robust and accessible alternative for vulnerability management, especially in educational environments, research laboratories, and security testing scenarios.

From an academic perspective, although this laboratory exercise represents a relatively simple scenario, it provides me a valuable opportunity to understand how vulnerability scanners detect security weaknesses and how these findings can be analyzed and validated in practice.
